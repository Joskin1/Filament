# Blog Post Project

This project is a blog platform built using the TALL stack (Tailwind CSS, Alpine.js, Laravel, and Livewire). It includes Jetstream for authentication and dashboard features and Filament for the admin and editor panel.

## Features

- **User Authentication:** Managed by Laravel Jetstream.
- **Admin & Editor Panel:** Built with Filament for managing posts and categories.
- **Blog Posts:** Users can view blog posts.
- **Categories:** Posts can have multiple categories.
